# Frontend Web

## Tecnología

- SPA vanilla JS sin frameworks
- Vite 8 como bundler + HMR
- Hash routing (`#/ruta`)

## Páginas

| Ruta | Página | Descripción |
|---|---|---|
| `#/` | Inicio | Landing, estado actual, últimos cambios del devlog |
| `#/devlog` | Devlog | Historial de cambios colapsable por fecha |
| `#/game` | Juego | Motor 3D con controles completos |
| `#/dev` | Desarrollo | Herramientas de modding (vacío por ahora) |
| `#/docs` | Documentación | Documentación del proyecto |

## Archivos

`src/client/` — router, pages, components, styles
